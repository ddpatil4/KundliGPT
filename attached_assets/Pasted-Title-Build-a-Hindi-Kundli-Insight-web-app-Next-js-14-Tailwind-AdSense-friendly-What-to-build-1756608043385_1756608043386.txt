OpenAI Integration

Install: npm i openai zod dayjs

Env: Use process.env.OPENAI_API_KEY

Model: gpt-4o-mini (or a capable text model) via Responses API

Create /api/interpret/route.ts (Next.js App Router) that:

Validates input with zod:

{
  name: string.min(1),
  birthDate: string,   // YYYY-MM-DD
  birthTime: string,   // HH:mm (24h)
  tzOffset: number,    // e.g. 5.5
  city?: string,
  lat?: number,
  lon?: number
}


Builds a system prompt to keep tone helpful & cautious (Hindi output)

Builds a user prompt injecting the fields; you may calculate a simple “demo” tag like contextNote: "Ephemeris not used; insights are generalized" to be transparent

Calls OpenAI Responses API and returns json: { ok: true, resultHtml } where resultHtml contains semantic HTML with the required sections
